# Riesgo climático en la tasación inmobiliaria

## Descripción
Este proyecto analiza el impacto del cambio climático (temperatura y precipitaciones) sobre el precio de la vivienda en España, utilizando técnicas de modelización estadística y machine learning.

## Objetivo
Evaluar si variables climáticas influyen significativamente en el precio por metro cuadrado y mejorar la capacidad predictiva mediante modelos avanzados.

## Datos utilizados
- Tasaciones oficiales (Ministerio de Vivienda)
- Datos macroeconómicos (Banco de España)
- Datos climáticos históricos y proyectados
- Dataset inmobiliario (Idealista)

## Tecnologías
- Python
- Pandas / NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Metodología
1. Limpieza y preparación de datos
2. Análisis exploratorio (EDA)
3. Feature engineering (variables climáticas)
4. Modelización:
   - Regresión lineal
   - SARIMAX
   - Random Forest
   - AdaBoost
5. Evaluación con métricas:
   - RMSE
   - R²

## Resultados
- AdaBoost muestra mejor capacidad predictiva frente a modelos lineales
- Las variables climáticas tienen impacto significativo en ciertas regiones

## Conclusiones
El cambio climático influye en la valoración inmobiliaria, especialmente en zonas con mayor exposición térmica.
